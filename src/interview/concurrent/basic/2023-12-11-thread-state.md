





| 线程状态      | 解释                                                     |
| ------------- | -------------------------------------------------------- |
| NEW           | 尚未启动的线程状态，即线程创建，还未调用start方法        |
| RUNNABLE      | 就绪状态（调用start，等待调度）+正在运行                 |
| BLOCKED       | 等待监视器锁时，陷入阻塞状态                             |
| WAITING       | 等待状态的线程正在等待另一线程执行特定的操作（如notify） |
| TIMED_WAITING | 具有指定等待时间的等待状态                               |
| TERMINATED    | 线程完成执行，终止状态                                   |
